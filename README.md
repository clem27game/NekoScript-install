
# NekoScript

## Installation 

Pour installer NekoScript, exécutez simplement :

```bash
neko-script télécharger
```

## Utilisation

### Exécuter un script
```bash
neko-script run mon_script.neko
```

### Publier un package
```bash
neko-script publish mon_package.neko
```

### Importer une librairie
```bash
neko-script librairie nom_librairie.neko
```
